# Data Sources

This document catalogs all sources used in compiling the Headbangers Ball dataset, along with reliability ratings and usage notes.

## Primary Sources

### 1. EpisodeHive
- **URL**: https://episodehive.com/tv-shows/headbangers-ball/all
- **Reliability**: ⭐⭐⭐⭐ (High)
- **Content Coverage**:
  - Episode air dates
  - Host information
  - Music video listings (artist + song)
  - Episode descriptions
- **Usage**: Primary source for episode listings and video catalogs
- **Notes**: Most comprehensive single source for episode-by-episode breakdowns
- **Citation Format**: `episodehive.com`

### 2. TheTVDB
- **URL**: https://thetvdb.com
- **Reliability**: ⭐⭐⭐⭐⭐ (Very High)
- **Content Coverage**:
  - Episode air dates
  - Episode numbers and seasons
  - Episode descriptions
  - Host information
- **Usage**: Cross-verification of air dates and episode metadata
- **Notes**: Community-maintained database with rigorous standards
- **Citation Format**: `thetvdb.com`

### 3. IMDb (Internet Movie Database)
- **URL**: https://www.imdb.com
- **Reliability**: ⭐⭐⭐⭐ (High)
- **Content Coverage**:

  - Host credits
  - Guest appearances
  - Some video listings
- **Usage**: Verification of episode dates and guest information
- **Notes**: May have incomplete video listings; best for host/guest data
- **Citation Format**: `imdb.com`

### 4. Wikipedia - Headbangers Ball
- **URL**: https://en.wikipedia.org/wiki/Headbangers_Ball

  - Revival information
- **Usage**: Establishing general timeline and host tenures

- **Citation Format**: `wikipedia.org`

### 5. Next-Episode

## Source Priority Hierarchy

When conflicting information appears, use this priority order:

1. **Direct video evidence** (YouTube archives of actual episodes)
2. **TheTVDB** (for dates and episode numbers)
3. **EpisodeHive** (for video listings)
4. **IMDb** (for host/guest information)
5. **Wikipedia** (for general timeline)
6. **Multiple secondary sources in agreement**
7. **Single secondary source** (flagged as needing verification)

## Verification Requirements

### Minimum Citation Standards

- **Episode dates**: Require 2+ sources OR 1 primary source + video evidence
- **Host information**: Require 2+ sources
- **Video listings**: Require 1 primary source OR video evidence
- **Special segments**: Require 1+ sources with description

### Confidence Levels

Each data entry should have an implicit confidence level based on sources:

- **High Confidence** (✓✓✓): 3+ sources or video evidence
- **Medium Confidence** (✓✓): 2 sources in agreement
- **Low Confidence** (✓): 1 source only (mark for verification)
- **Unverified** (?): Anecdotal/needs sources
